# LensPrime MVP

## Goal

Allow a user with little or no SQL knowledge to import data, define relationships, and generate SQL queries through a guided interface.

---

## Required Features

### Data Import

* Import CSV files
* Create SQLite workspace database
* Discover tables and columns

### Schema Detection

* Detect likely primary keys
* Detect likely relationships
* Allow relationship review and editing

### Query Builder

* Select metric
* Select grouping
* Apply filters
* Select sort order
* Apply result limits

### Query Execution

* Generate SQL
* Execute SQL
* Display results

### User Interface

* Upload data through a graphical interface
* Review generated SQL
* Execute queries from the interface

---

## Not Required for MVP

* User accounts
* Cloud hosting
* AI-generated SQL
* Tableau integration
* Power BI integration
* Dashboard generation
* Multi-user support
* Data warehousing

---

## MVP Success Criteria

A user can:

1. Upload CSV files
2. Confirm relationships
3. Build a query using guided selections
4. Execute the query
5. View the results
6. Export the results

without writing SQL manually.
